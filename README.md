# Clickgame - Funny game
## Clone project
	### SSH
	`git clone git@github.com:at-dongvd/Clickgame.git`

	### HTTPS
	`git clone https://github.com/at-dongvd/Clickgame.git`

## run project
	### install package in file package.json(socket.io, moniker, ...)
	`npm install`

	### open cmd
	`node server.js`

	### go to link
	http://localhost:8888
